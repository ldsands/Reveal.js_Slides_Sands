## Goals of the Graduate Program (from the [Graduate Student Manual pg. 3](https://clas.uiowa.edu/sociology/sites/clas.uiowa.edu.sociology/files/9_28_20%20Graduate%20Student%20Manual.pdf)) 1. The attainment of general competence in theory, methods, and the substantive literature of sociology, and 1. The attainment of special competence within an area of concentration or "major" plus substantial competence in a "minor" field.
## So Many Sources - As you now know you will be reading hundreds of article and books per semester - You are responsible for the information you glean from these sources - Knowing who wrote what becomes more and more important (especially for area exams)
## Why Reference Management - If you keep these sources organized you will be able to: - Easily find sources you're familiar with - Spend less time finding the same piece of information "again"
source: Samet Kurtkus - https://unsplash.com/photos/HKFgPPPLdyI
## Paper - Many prefer paper - Paper is easier for some to: - Highlight - Take notes - Many sources are only available in print
## Electronic - Electronic sources: - can be indexed and searched - can create automated bibliographies - can be accessed anywhere - some sources are only available electronically
## Paper and Electronic - Electronic Metadata: - can be indexed and searched - can create automated bibliographies - can be accessed anywhere - some sources are only available electronically - Paper Sources: - Can be "tagged" in Electronic software - Highlight - Take notes - Many sources are only available in print
## Software Options (More options can be compared [on Wikipedia](https://en.wikipedia.org/wiki/Comparison_of_reference_management_software)) - [Citavi](https://en.wikipedia.org/wiki/Citavi) (Windows Only, Free with few limitations) - [EndNote](https://en.wikipedia.org/wiki/EndNote) (Cross-Platform, One Time Purchase $299) - The [university provides EndNote for free](https://its.uiowa.edu/endnote) to students and faculty - [Mendeley](https://en.wikipedia.org/wiki/Mendeley) (Cross-Platform, $5 to $15 a month) - [Paperpile](https://en.wikipedia.org/wiki/Paperpile) (Uses Chrome, #3/month) - [Papers](https://en.wikipedia.org/wiki/Papers_(software)) (Cross-Platform, $3 or $5/month) - [Zotero](https://en.wikipedia.org/wiki/Zotero) (Cross-Platform, Freeish more details later)
## Components of Using Zotero - Zotero and Other Servers - Storage - Citation Storage (uses your Zotero account for this storage) - Attachment Storage (two options) - 300 mg Free - 2 GB $20/year - 6 GB $60/year - Unlimited $120/year - WebDAV pCloud is the best option - 10 GB free - Zotero Desktop Application - Interface with your citations and files - Organization (folders and tags) - Using your files - Syncing your citations and files - Sharing citations and files - Notes - Searching in tags, folders, and files - Zotero Browser Extension - Extract citation and files from internet site - Zotero Word Processor Add-In (MSWord and LibreOfficeWord)
## Zotero: Desktop Application Walk-Through - Library sidebar - Folders - Duplicate Items - Unfiled Items - Trash - Group Sharing - Feeds (for RSS) - Tags - can you colors - Main window - customize columns (date, extra, etc.) - Details/Editing Sidebar - Info, notes, tags, related - Buttons and Other Main Window UI Highlights - Sync button - Search Bar/Advanced Search ("Title, Creater, Year" | "All Fields & Tags" | "Everything") - Add Item(s) by Identifier (using a DOI or similar Identifier e.g. https://doi.org/10.1146/annurev-soc-071811-145503) - Other - Collections, "ctrl" to view
## Zotero: Desktop Application Walk-Through Continued - Settings - View -> Show Items from Subcollections - Edit -> Preferences - General -> Snapshots - Sync -> Data Syncing | File Syncing - Search -> Full-Text Cache - Export -> Quick Copy - Cite -> Styles -> Style Manager | Cite -> Word Processors -> MS Word/LibreOffice (JRE is needed for LibreOffice) - Other Stuff - Feeds
## Zotero: Browser Extension - Retrieving Citations - Google Scholar ([example](https://scholar-google-com.proxy.lib.uiowa.edu/scholar?hl=en&as_sdt=0%2C16&q=Status%3A+Insights+from+Organizational+Sociology&btnG=)) - " option - Zotero Extension Option - Publisher Option - Zotero Feature Right Click -> "Find Available PDF" - Web of Science ([example](https://apps-webofknowledge-com.proxy.lib.uiowa.edu/Search.do?product=WOS&SID=7CBxsCVxfBq9FtzaIOV&search_mode=GeneralSearch&prID=cf20a5e6-cfbb-4d64-9d6b-d25c5490bc3f)) - Citation Report/Times Cited/Cited References - UILink/"Look Up Full Text" - No Books? - INFOHawk+ ([example](https://search.lib.uiowa.edu/primo-explore/search?query=any,contains,A%20member%20saved%20is%20a%20member%20earned%3F%20The%20recruitment-retention%20trade-off%20and%20organizational%20strategies%20for%20membership%20growth&tab=default_tab&search_scope=default_scope&vid=01IOWA&lang=en_US&offset=0)) - "Full text may be available" - "Download PDF" - Send to -> "EXPORT RIS" - Elsewhere ([example link1](https://scholar-google-com.proxy.lib.uiowa.edu/scholar?hl=en&as_sdt=0%2C16&q=From+Max+Weber%3A+Essays+in+Sociology+1958&btnG=), [example link 2](https://search.lib.uiowa.edu/primo-explore/fulldisplay?docid=01IOWA_ALMA21421640050002771&context=L&vid=01IOWA&lang=en_US&search_scope=default_scope&adaptor=Local%20Search%20Engine&tab=default_tab&query=any,contains,From%20Max%20Weber:%20Essays%20in%20Sociology%201958&offset=0)) - Google Books vs Amazon vs "Other means"
## Zotero: Word Processor Citations - Works in both MS Word and LibreOffice Writer (Requires JRE) - Document Preferences (use fields when possible) -> ASA 6th edition - Multiple (example "(Fukuyama 2001; Glanville 2014; Putnam 2000)") - Suppress Author, Page Number, Change Citation Group - Copy plain text citations (e.g. Markdown and Pandoc, etc.) citation shortcut: "ctrl+shift+a"; bibliography shortcut: "ctrl+shift+c"
## Zotero: Other Useful Features - Plugins: Zotfile (file management); Google Scholar Citations for Zotero; Zotero DOI Manager; Zotero OCR (optical character recognition); Better BibTeX with VS Code plugin; Go [here](https://www.zotero.org/support/plugins) for more - Collection Timelines: Select a Collection -> Tools -> Create Timeline - Generate Reports: Select items desired -> Right Click -> Generate Report from Items - Export Items: Select items desired -> Right Click -> Export Items
